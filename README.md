# ⚡ VerifiedSXO - Quick Deploy

## Deploy in 60 Seconds

### 1. Extract & Navigate
```bash
unzip verifiedsxo-v1.0.zip
cd verifiedsxo/frontend
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Deploy to Vercel
```bash
npx vercel
```

### 4. Add Your Anthropic Key (Optional but recommended)
```bash
npx vercel env add ANTHROPIC_API_KEY
```
Enter your key when prompted.

### 5. Connect Domain
In Vercel Dashboard → Settings → Domains → Add `verifiedsxo.com`
